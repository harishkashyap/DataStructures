DataStructures

DataStructures library contains the following basic data structures:

  • Linked Lists
  • Binary Trees
  • Stacks
  • Queue

Usage

To run the example project; clone the repo, and run pod install from the Example directory first.

Requirements

Installation

DataStructures is available through CocoaPods, to install it simply add the following line to your Podfile:

pod "DataStructures"

Podfile

platform :ios, '7.0'
pod "DataStructures", "~> 0.1.1"
